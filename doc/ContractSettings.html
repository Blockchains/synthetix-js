Class: ContractSettings

ContractSettings(provider, signer, fromAddressIndex, networkId)

new ContractSettings(provider, signer, fromAddressIndex, networkId)

Parameters:
Name Type Description
provider Object ethers.js provider object - default ethers.providers.getDefaultProvider()
signer Object one of "Metamask"|"Trezor"|"Ledger" or a custom ethers.js compatible signer. Use Metamask for Dapp browser support
fromAddressIndex Number index of a generated from address for a hardware wallet (Metamask/Dapp browser will always use 0 as address can be selected in UI)
networkId Number default 1 - mainnet, also supports 42 (Kovan)
